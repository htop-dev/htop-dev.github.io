htop - an interactive process viewer

This is htop, a cross-platform interactive process viewer. It is a text-mode application (for console or X terminals) and requires ncurses.

Latest release.

What's new in htop

The latest releases in htop include delay accounting metrics for Linux, Solaris/Illumos/OpenIndiana and DragonFly BSD support, as well as many other features and bugfixes. Check out the changelog for more details and credits!

Since version 2.0, htop is now cross-platform! Check out the video and slides of Hisham's presentation at FOSDEM 2016 about how this came to be. This release includes code supporting Linux, FreeBSD, OpenBSD, DragonFly BSD and MacOSX.
